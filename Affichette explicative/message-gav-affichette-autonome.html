MESSAGE GAV

Scannez ce QR code pour ouvrir l’application sur votre téléphone, puis ajoutez-la à l’écran d’accueil pour l’utiliser comme une vraie app.

Accès rapide

Lien direct :
https://gend-231817.github.io/Message-GAV/

Installation

iPhone
Ouvrir le lien dans Safari, toucher Partager, puis choisir Sur l’écran d’accueil.
Android
Ouvrir le lien dans votre navigateur, toucher le menu ⋮, puis choisir Installer l’application ou Ajouter à l’écran d’accueil.
NEOGEND
Saisissez l’adresse https://gend-231817.github.io/Message-GAV/ sur Firefox, toucher le menu ⋮, puis choisir Ajouter l’application à l’écran. (Plusieurs tentatives peuvent être nécessaires)
Si cette fiche est consultée depuis un téléphone, le lien ci-contre est cliquable et permet d’ouvrir directement l’application sans scanner le QR code.
L’application apparaîtra ensuite comme une icône sur votre téléphone. Usage professionnel
